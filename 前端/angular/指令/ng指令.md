

### 建立新專案
```properties
ng new my-app
```

### ⭐開啟伺服器(--open:打開瀏覽器)
```properties
ng serve --open
```

### Angular更方便引用庫的方式 (約同npm install)
```properties
ng add
```

### 檢查可更新的angular相關庫
```properties
ng update
```

### ⭐更新cli至穩定版本，https://angular.io/cli/update
```properties
ng update @angular/cli @angular/core
```
